Self-Supervised Pre-trained model for Underwater Image Segmentation

Fall 2024

Current approaches to underwater image segmentation primarily focus on either fully training models on a single dataset or employing transfer learning techniques by pretraining encoders on terrestrial image datasets. However, these methods often struggle with domain adaptation challenges inherent to underwater environ- ments. To address these limitations, our methodology leverages the SimCLR framework with a ResNet-50 backbone to pretrain a model capable of learning robust feature embeddings in an unsupervised manner.

• Pretrained using SimCLR on the UFO dataset with advanced data augmentation for underwater image features.
• Levarged SimCLR's constrastive loss to achieve minimum distance between positive features in feature space.
• Exploited U-Net's Encoder-Decoder architecture for fine-tuning for SUIM dataset.
• Fine-tuned U-Net on the SUIM dataset, achieving 0.70 pixel accuracy (baseline: 0.67) using a hybrid loss function, that combines dice loss(attention to thin features) and cross entropy loss(minimize pixel to pixel loss).

  • OpenCV | Python | Deep learning
